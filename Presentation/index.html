Refit

The automatic type-safe REST library for Xamarin and .NET

By Paul Betts (github): https://github.com/paulcbetts/refit

Why?

  • Fast and easy to start using
  • Flexible
  • Strongly Typed
  • Removes all the plumbing of
    using HttpClient
  • Supports multiple platforms
  • Good documentation
  • Attribute all the things!

When?

Consuming REST services

When you want to consume REST services in a type-safe way.

Refit currently supports the following platforms:

  • Xamarin.Android
  • Xamarin.Mac
  • Desktop .NET 4.5
  • Windows Phone 8
  • Silverlight 5
  • In the works: Xamarin.iOS
